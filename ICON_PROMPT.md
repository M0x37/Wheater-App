# Weather App Icon Design Prompt

## App Overview
**Name**: Weather App  
**Type**: Minimalist weather application  
**Platform**: Android mobile app  
**Design Style**: Dark theme, monospace typography, clean interface

## Core Features
- Real-time weather data display
- Location search functionality  
- Multiple forecast views (current, today, 7-day)
- Temperature, wind, humidity information
- Time-based weather periods (morning, afternoon, evening, night)

## Design Requirements

### Visual Style
- **Minimalist and modern** - Clean, uncluttered design
- **Dark theme friendly** - Must look great on black/dark backgrounds
- **Professional appearance** - Suitable for daily use
- **Highly recognizable** - Clear weather app identification at small sizes

### Color Palette
- **Primary**: Dark mode colors (black, dark gray)
- **Accent**: Blue tones for cold/winter weather
- **Secondary**: White/gray for text and details
- **Optional**: Warm colors for temperature indicators

### Icon Elements to Consider
- Weather symbols (sun, clouds, rain, snow)
- Temperature indicators (thermometer, degree symbol)
- Location/map elements
- Time/clock elements for forecast periods
- Clean geometric shapes

## Technical Specifications

### Format Requirements
- **Primary**: SVG format for scalability
- **Sizes**: Must work at 32x32, 48x48, 72x72, 96x96, 144x144, 192x192 pixels
- **Background**: Transparent background
- **Resolution**: Crisp at all sizes, no blurring

### Design Constraints
- **Square aspect ratio** - Standard app icon format
- **No text** - Pure visual communication
- **Simple shapes** - Avoid complex details that get lost at small sizes
- **High contrast** - Must be visible on both light and dark backgrounds

## Creative Direction

### Concept Ideas
1. **Minimalist Weather Symbol** - Simple sun/cloud combination with temperature element
2. **Geometric Weather** - Abstract representation using circles, triangles, lines
3. **Location + Weather** - Subtle location pin with weather indicator
4. **Time-based Weather** - Clock-like element with weather symbols

### Mood & Feeling
- **Calm and reliable** - Users check weather daily, needs to feel dependable
- **Modern and tech-focused** - Reflects the app's clean, data-driven approach
- **Approachable** - Not intimidating, easy to understand
- **Precise** - Matches the app's accurate weather data presentation

## Inspiration References
- Modern weather apps with minimalist design
- Material Design icon guidelines
- iOS weather app icon simplicity
- Professional meteorological service symbols

## Deliverables
- Main app icon in SVG format
- Alternative variations (if applicable)
- Brief explanation of design concept
- Color palette used in the design

## Success Criteria
- Instantly recognizable as a weather app
- Looks professional and modern
- Works perfectly at all required sizes
- Matches the app's dark, minimalist aesthetic
- Stands out in a crowded app drawer

---

**Design Brief Summary**: Create a minimalist, modern weather app icon that works beautifully in dark theme, is highly recognizable at small sizes, and reflects the app's clean, data-driven approach to weather information.
